# Linux Jobs

Yes, we love Linux community!
Job board for Linux Administrators.

## Clone the repository
```
git clone https://github.com/thiagoaugusto/linuxjobs.git
```

## Install dependencies
```
bundle install
```

## Create database
```
rake db:create
rake db:migrate
rake db:seed
```

## Run tests
```
rails test
```

## Run server
```
rails server
```
